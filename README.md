
# Pomodoro Focus Timer

A modern, feature-rich Pomodoro timer web application designed to help users focus on work or study. Built with vanilla JavaScript using Web Components, it offers a clean interface, multiple themes, and customizable sessions.

## ✨ Features

- **Pomodoro Timer:** alternate between focused work sessions and refreshing break sessions.
- **Customizable Sessions:** Set custom durations for both work and rest periods.
- **Lock Mode:** A full-screen overlay with a timer and a motivational quote to minimize distractions.
- **Sound & Vibration Alerts:** Optional audio and vibration notifications at the end of each session.
- **Dark/Light Theme:** Switch between a light and a dark theme for visual comfort.
- **Responsive Design:** Works seamlessly on both desktop and mobile devices.
- **Framework-Free:** Built with modern web standards (Web Components, ES Modules) for a lightweight and fast experience.

## 🚀 Getting Started

Simply open the `index.html` file in your browser to start using the application.

## 🛠️ Built With

-   HTML5
-   CSS3 (with CSS Variables for theming)
-   Modern JavaScript (ES6+, Web Components)
